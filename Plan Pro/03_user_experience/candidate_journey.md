# Candidate Journey (MVP)

## Owns
- Candidate-facing lifecycle flow and stage objectives.

## Excludes
- Screen-level UI specs (see `ui_specification.md`).
- Notification copy text (see `notifications.md`).

## Stages
1. Account creation and verification.
2. Consent capture.
3. CV upload/import and profile bootstrap.
4. High-impact field confirmation and matchability.
5. Match list + invite details.
6. Q&A session completion.
7. Submission confirmation and status tracking.
8. Interview scheduling and offer status surfaces.

## Stage Exit Rules
- Candidate becomes matchable only after required fields are valid.
- Invite can start Q&A only if invite is non-terminal and non-expired.
- Submission exit requires all required questions valid.
